# AI-AGENCY-GAME
An AI Agency game that teaches you how to run and manage an AI agency.

## Description
Build and manage your own AI agency in this interactive text-based simulation game. Make strategic decisions about hiring, client work, marketing, and training to survive 52 weeks (1 year) in the competitive AI industry.

## Installation

```bash
npm install
```

## How to Play

```bash
npm start
```

## Game Features

- **Client Management**: Take on client projects to earn revenue
- **Team Building**: Hire employees to increase your capacity
- **Marketing**: Invest in marketing to boost your reputation
- **Training**: Improve your team's skills and reputation
- **Strategy**: Balance cash flow, reputation, and growth

## Game Mechanics

- **Starting Resources**: $10,000 and 1 employee
- **Client Projects**: Worth $5,000 each, requires good reputation
- **Employee Costs**: $3,000 to hire, $2,000/week salary
- **Capacity**: Each employee can handle 2 clients simultaneously
- **Goal**: Survive 52 weeks with positive cash flow

## Testing

```bash
npm test
```

## Tips for Success

1. Don't hire too many employees too quickly
2. Maintain a good reputation to attract clients
3. Balance revenue and expenses carefully
4. Invest in marketing when reputation is low
5. Always have enough cash for operating costs
